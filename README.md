# Reconocimiento de Patrones (2026-2)

Repositorio dedicado al curso de **Reconocimiento de Patrones** para la licenciatura en **Ciencia de Datos** de la **UNAM**, impartido en el **Instituto de Investigaciones en Matemáticas Aplicadas y en Sistemas (IIMAS)**.

Este espacio contiene implementaciones, ejercicios y proyectos enfocados en:

* Preprocesamiento de imágenes.
* Extracción de características.
* Modelos de clasificación.
* Diseño de sistemas de reconocimiento de patrones.

---

## Prácticas Realizadas

### [RP_P1] Preprocesamiento de datos y extracción de características

En esta práctica se exploraron las distintas etapas del procesamiento de datos de diversa índole,
abarcando desde datos tabulares hasta imágenes y señales electrocardiográficas. Este proceso es
fundamental para la creación y diseño de sistemas de reconocimiento de patrones.

### [RP_P2] Extracción de características mediante LBP

En esta práctica se implementó el método de Patrones Binarios Locales (LBP) para la extracción
de características. A partir del análisis de los histogramas LBP generados, se compararon distintos
modelos de clasificación y se evaluó su exactitud.
